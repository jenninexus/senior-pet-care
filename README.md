# Senior Pet Care Tracker

A free, open-source health tracking system for senior cats (and adaptable for dogs).

Built from real-world experience caring for **Alice**, an 18-year-old cat with CKD and hyperthyroidism — managed at home with vet guidance, daily logs, and a lot of love.

---

## What's Included

| File | Purpose |
|------|---------|
| [daily-log.md](daily-log.md) | Day-by-day health observations |
| [weekly-weight-log.md](weekly-weight-log.md) | Weekly weight tracking with trend notes |
| [quality-of-life.md](quality-of-life.md) | Monthly QoL scoring (HHHHHMM scale) |
| [vet-visit-history.md](vet-visit-history.md) | Full vet visit timeline template |
| [medications.md](medications.md) | Medication + supplement tracker |
| [red-flags.md](red-flags.md) | Emergency reference card |
| [end-of-life-planning.md](end-of-life-planning.md) | Compassionate EOL planning guide |
| [example/](example/) | Alice's actual tracker (anonymized example) |

---

## Who This Is For

- Anyone with a senior cat (10+ years) managing chronic illness
- Especially useful for: CKD, hyperthyroidism, diabetes, heart disease, arthritis
- Designed to be printed or used digitally — no app required
- Works alongside vet care, not instead of it

---

## How to Use

1. **Copy the templates** — duplicate the files in this repo for your pet
2. **Fill in daily** — even 2 minutes of observation data adds up over weeks
3. **Bring the logs to vet visits** — your observations help the vet see patterns they can't see in a 20-minute appointment
4. **Share with family** — multiple caregivers can stay coordinated

---

## The Example

The `example/` folder contains a complete tracker based on Alice — an 18-year-old cat with dual CKD + hyperthyroidism diagnoses, managed at home with sub-Q fluids, Phos-Bind, B12 injections, and methimazole.

Her weight has been stable at 8.52 lbs since November 2024 — a real win for a cat her age with her conditions.
